# Activité Pratique N°6

Cette application Spring Boot intègre Kafka pour l'envoi et la réception de messages, tout en utilisant des principes de programmation fonctionnelle au sein du framework Spring Boot. Le projet englobe à la fois les fonctionnalités de producteur et de consommateur Kafka.
